Fonctionnement du sommaire


L'afficher

Pour afficher le sommaire auppuyez sur l'endroit où se trouve trois rectangles l'un en dessous de l'autre, ceci permettra de faire glisser un volé contenant le sommaire.


S'y reprer

Une fois dans le sommaire, pour se reprer il suffit de cliquer sur les premiers mot compris dans le question que vous vous posez.

Par exemple, si vous souhaiter savoir "Comment soigner les champignons chez les chats?", vous pouvez faire ce type de raisonnement:

Le premier mot est "Comment" Vous irez donc dans le rubrique "Comment" et ferez défiler le menu jusqu'à trouver le prochain mot "soigner" et irez dans cette rubrique et chercherez le prochain mot "les champignons" et vous tomberez sur la fin de la question "les champignons chez les chats".


Ce type de raisonnenment est celui que je vous recommande d'utiliser pour vous reprer dans cette page.


Le but de cette page

Le but de cette page est de vous permettre de trouver les réponses à vos question par rapport au chats.


Il y a une petite rubrique se du nom de Pour retrouver sont chat à l'aide de sont téléphone qui vous propose quelques produits qui sont des GPS pour ne plus perdre sont chat.
